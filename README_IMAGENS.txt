Como usar no código:
No game.js, cada item aceita uma chave img.
Exemplo:
{ id:'bike', name:'Bicicleta', emoji:'🚲', img:'assets/veiculos/bicicleta.png', buy:900, monthly:35, status:1, speed:1 }

Sites bons para buscar:
1) Kenney — assets de jogo CC0/public domain. Bom para ícones, veículos, interfaces, prédios e objetos.
2) Quaternius — modelos 3D low poly CC0. Bom para casas, prédios, veículos, personagens e objetos 3D.
3) OpenGameArt — filtre por CC0/public domain. Confira a licença de cada asset individualmente.
4) itch.io — pesquise por "cc0", "free game assets", "vehicles cc0", "furniture cc0", "icons cc0".

Pesquisas úteis:
- furniture cc0 game assets
- vehicle cc0 game asset
- house building cc0 asset
- low poly car cc0
- jewelry icon cc0
- office furniture cc0
- city buildings cc0

Dica: use PNG/WebP para 2D e GLB/FBX/OBJ só se depois transformar o jogo para engine 3D.
