Como testar

1
Clique em Iniciar jogo
Crie seu personagem e escolha cidade/área inicial.
2
Faça contratos pequenos
No começo, contratos grandes ficam bloqueados por habilidade e estrutura.
3
Salve seu progresso
O jogo salva no navegador de cada pessoa usando localStorage.
💾 Progresso dos seus amigos
Cada amigo terá o próprio save no navegador dele. Para testar online no GitHub Pages, isso já funciona sem servidor.

Dica: para ranking online ou save em nuvem, precisa usar backend como Firebase/Supabase/Google Sheets API. Excel local não salva dados automaticamente de vários jogadores.
